## 安装
当你在使用`claudecode`或者`opencode`的时候，可以直接通过git链接安装技能，例：
```
通过https://github.com/mailzwj/skills/tree/main/ltx-prompt-designer安装技能
```

## 用法
当你的claudecode或者opencode提示技能安装成功后，最好重启你的工具，然后就可以正常使用该技能了，例：
```
/ltx-prompt-designer 生成一段海边散步的视频
```
触发短语包括："生成 LTX 提示词"、"LTX 视频 prompt"、"AI 视频生成"、"写个视频提示词"、"帮我写个 LTX prompt"、"LTX 视频脚本"、"视频场景描述"等
