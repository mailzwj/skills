## 安装
当你在使用`claudecode`或者`opencode`的时候，可以直接通过git链接安装技能，例：
```
通过https://github.com/mailzwj/skills/tree/main/ace-caption-creator安装技能
```

## 用法
当你的claudecode或者opencode提示技能安装成功后，最好重启你的工具，然后就可以正常使用该技能了，例：
```
/ace-caption-creator 生成音乐
```
你可以参考以下技能触发词："生成音乐"、"创作歌曲"、"AI作曲"、"写个caption"、"生成caption"、"音乐提示词"、"ACE音乐"等
