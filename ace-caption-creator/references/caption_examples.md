## 提示词构建公式

### 基础公式
```
[风格] + [情绪] + [乐器] + [节奏] + [特色]
```

### 进阶公式
```
[主风格] + [子风格/融合] + [情绪变化] + [核心乐器] + [辅助音色]
+ [节奏/BPM] + [音色质感] + [参考风格] + [使用场景]
```

### 示例解析
```
"Melodic techno (主风格) with progressive elements (子风格),
hypnotic and building (情绪变化), featuring arpeggiated synth
leads (核心乐器) over deep sub bass (辅助音色), 126 BPM with
steady groove (节奏), analog warmth and digital precision (音色质感),
influenced by Prydz and Anyma (参考), for festival mainstage (场景)."
```

## 自检清单

创建caption后，检查是否包含：

- [ ] 明确的音乐风格
- [ ] 清晰的情绪方向
- [ ] 主要乐器配置
- [ ] 节奏速度指示
- [ ] 音色质感描述
- [ ] 避免矛盾元素
- [ ] 符合目标用途
- [ ] 长度适中（50-150词）
