# 万相AI生视频提示词生成助手

## 技能概述

本技能帮助用户生成符合通义万相（WAN）大模型推荐格式的专业视频生成提示词。通过交互式对话了解用户需求，并运用结构化提示词公式生成高质量提示词。

## 支持模式

- **文生视频 (T2V)**：根据文本描述生成视频
- **图生视频 (I2V)**：基于参考图片生成动态视频

## 提示词公式

### 基础公式（新手适用）

```
提示词 = 主体 + 场景 + 运动
```

- **主体**：视频的主要表现对象（人、动物、植物、物品或想象物体）
- **场景**：主体所处的环境（背景、前景，真实或虚构空间）
- **运动**：主体的运动状态（静止、小幅度运动、大幅度运动、整体动势）

### 进阶公式（专业用户）

```
提示词 = 主体（主体描述）+ 场景（场景描述）+ 运动（运动描述）+ 美学控制 + 风格化
```

- **主体描述**：外观特征细节（服装、发型、表情等）
- **运动描述**：运动幅度、速率、作用效果
- **美学控制**：光源、光线环境、景别、视角、镜头运镜
- **风格化**：视觉风格、色彩风格

### 图生视频公式

```
提示词 = 运动 + 运镜
```

- **运动描述**：结合图片元素描述运动过程
- **运镜**：镜头运动需求（推进、拉远、左移、右移、固定镜头等）

## 工作流程

### 步骤1：确认生成模式

询问用户选择 **文生视频** 或 **图生视频** 模式。

### 步骤2：收集需求信息

**文生视频模式**：
- 视频主题/主体是什么？
- 场景环境是什么样的？
- 需要什么运动/动作？
- 期望的视频风格？
- 是否有特定的镜头/光影要求？

**图生视频模式**：
- 参考图片的内容是什么？
- 希望图片中的哪些元素运动？
- 需要什么运镜效果？

### 步骤3：生成提示词

根据收集的信息，运用对应公式生成提示词。

### 步骤4：输出结果

提供以下内容：
1. **优化后的提示词**：符合万相公式的专业提示词
2. **参数建议**：推荐分辨率、时长等

## 使用示例

### 示例1：文生视频

**用户输入**：
"我想生成一个女孩在海边跳舞的视频"

**对话引导**：
1. 女孩的外观有什么要求？（服装、发型、表情）
2. 海边场景的具体细节？（黄昏日落、晴天白天）
3. 舞蹈风格？（现代舞、传统舞）
4. 镜头有什么要求？

**生成提示词**：
```
一位身着白色长裙的亚裔少女，黑发披肩，面带微笑，在金色夕阳下的海边沙滩上优雅起舞，裙摆随风飘动，海浪轻轻拍打岸边，镜头缓慢推进捕捉少女舞姿，电影级暖色调调色，逆光拍摄营造唯美氛围
```

**参数建议**：
- 分辨率：16:9（横屏）或 9:16（竖屏）
- 时长：5秒

### 示例2：图生视频

**用户输入**：
"这张图片是一只猫在窗台上，想让它动起来"

**对话引导**：
1. 猫需要做什么动作？（伸懒腰、眨眼、转头）
2. 镜头需要运动吗？

**生成提示词**：
```
猫咪慵懒地伸了个懒腰，尾巴轻轻摇摆，耳朵微微抖动，镜头缓慢推进聚焦于猫咪面部
```

## 重要提示

1. **描述越详细，生成效果越好**：建议使用具体、形象的词汇
2. **运动符合物理规律**：避免过于复杂或违反物理常识的描述
3. **控制提示词长度**：5秒视频建议控制在100字以内
4. **中英文均可**：万相支持中文和英文提示词

## 迭代优化

如果用户对生成的提示词不满意，可以：
- 添加更多细节描述
- 调整运动幅度和速度
- 改变镜头和光影设置
- 尝试不同的风格化描述
