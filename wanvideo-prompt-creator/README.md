## 安装
当你在使用`claudecode`或者`opencode`的时候，可以直接通过git链接安装技能，例：
```
通过https://github.com/mailzwj/skills/tree/main/wanvideo-prompt-creator安装技能
```

## 用法
当你的claudecode或者opencode提示技能安装成功后，最好重启你的工具，然后就可以正常使用该技能了，例：
```
/wanvideo-prompt-creator 视频生成
```
触发场景：(1)用户需要生成AI视频提示词 (2)想了解万相提示词公式 (3)寻求视频创意描述 (4)优化现有提示词
